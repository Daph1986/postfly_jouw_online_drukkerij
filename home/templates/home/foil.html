{% trans 'What is Inline Cold Foil?' %}

{% trans 'Cold foil makes it possible to provide the printing sheets with metallic effects' %} {% trans 'in a single printing run by means of real foil as a spot – so locally – applied.' %}

{% trans 'How does Inline Cold Foil work?' %}

{% trans 'The adhesive layer is applied to the first printing tower.' %} {% trans 'Using a standard printing plate,' %} {% trans 'the adhesive layer is printed on the paper exactly where the foil should go.' %} {% trans 'At the second printing tower, the printing sheet is pressed against the foil.' %} {% trans 'The foil attaches itself sharply and 100% permanently to the adhesive layer.' %} {% trans 'The contactless sheet transport with aerodynamic air tracks guarantees a scratch' %} {% trans 'and blemish-free print production.' %} {% trans 'Due to the very accurate fit,' %} {% trans 'the data for the foil can be formatted as data for an additional PMS color.' %} {% trans 'The CMYK color printing starts from the third printing tower.' %} {% trans 'The entire sheet can now be printed in full color: both the unprinted and the metallized part.' %} {% trans 'A dispersion varnish is applied as a protective layer.' %}
