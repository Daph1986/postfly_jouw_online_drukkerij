{% trans 'What is overprint?' %}

{% trans 'Sometimes bizarre things happen to your design, white texts disappear,' %} {% trans 'suddenly different colors come through your image.' %} {% trans 'And that while this was not visible at all or just visible on your design.' %} {% trans 'Such problems usually occur due to the incorrect use of overprint settings in the original layout.' %}

{% trans 'If 2 objects, both with a different color are placed on top of each other,' %} {% trans 'where they overlap, the color will be saved.' %}
{% trans 'The underlying color will not be visible.' %} {% trans 'This is not the case when using overprint.' %} {% trans 'The underlying will be visible here.' %}

{% trans 'Let’s illustrate this with examples.' %}

{% trans 'Example 1 WITHOUT overprint:' %}
{% trans 'The magenta square is no longer a complete square,' %} {% trans 'where cyan overlapped the magenta, there is now a recess.' %}

{% trans 'Example 2 WITH overprint:' %}
{% trans 'If we put the cyan surface in overprint,' %} {% trans 'the magenta surface will come through.' %} {% trans 'A full square is visible on both plates BUT the printing result will be different.' %} {% trans 'Where the colors overlap, a purple color can be seen. Cyan + Magenta = Purple' %}

{% trans 'This already explains a lot.' %} {% trans ' White text in overprint on a black surface will certainly go wrong,' %} {% trans 'because white is not a color which can be printed.' %} {% trans 'A white text should therefore always be saved' %} {% trans 'and should certainly not be put in overprint.' %} {% trans 'The white of your design is the paper.' %}

{% trans 'How can I check if overprint has been applied and use it if necessary?' %}

{% trans 'It is very easy to check whether overprint has been used.' %} {% trans 'In Indesign (2021),' %} {% trans 'click on View → Overprint Preview in the menu bar at the top.' %}

{% trans 'If you want to use the overprint options,' %} {% trans 'click in the menu bar on Window → Output → Attributes.' %} {% trans 'A window with the overprint functions appears.' %}
