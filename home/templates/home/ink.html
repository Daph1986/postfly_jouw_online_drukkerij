Ink coverage

{% endblock %} {% block content %}

Paper can only absorb a maximum percentage of ink. If more ink is supplied than paper can handle, you will end up with dirty smudges and streaks. The maximum ink coverage is 300%. You can easily check this yourself by adding up the values of the 4 printing colors. In the example below we create a new color in Indesign. If we add the 4 values together, we get at a percentage of 200%.
So there is no chance of streaks or spots due to an excess of ink.

How do you check this yourself?

In Indesign (2021) click in the menu bar on Window→ Output→ Separations Preview. Then you will see the window below.

At view you choose the option ‘separations’. With this option the values of the colors can be measured. Or you can choose the option “Ink Limit”. With this option you can see where there is an excess of ink.
